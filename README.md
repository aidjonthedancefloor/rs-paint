# RS-Paint

![rs-paint alpha logo](/icons/logo.png)

A lightweight image editor, written in Rust using GTK4.

## Features

- Basic Tools
    - Cursor (pan)
    - Pencil (+ various brushes and blending modes)
    - Eyedropper
    - Rectangle Select
    - Magic Wand
    - Fill
- Image Transformations
  - Flips
  - Rotations
  - Resizes
      - Cropping
      - Expansion
      - Scaling
- Import/Export (most common formats are supported)
- Tabs
- Layers
- Multi-Level Undo
